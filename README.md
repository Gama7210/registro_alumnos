# registro_alumnos

## Instrucciones rápidas:
- Importa el archivo `db.sql` en MySQL Workbench para crear la base de datos `registro_alumnos` y tablas.
- Edita `inc/db.php` y configura las credenciales de tu MySQL (host, usuario, contraseña).
- Coloca todo en tu servidor Apache (xampp htdocs) y accede a `http://localhost/registro_alumnos/index.php`.

- Archivos principales:
- `index.php` - página principal con navegación
- `grupos.php` - Registro de Grupos (interfaz principal)
- `alumnos.php` - Registro de Alumnos
- `registrados.php` - Lista de Alumnos Registrados
- `catalogos.php` - Configuración de catálogos (carreras, turnos)
- `db.sql` - script para crear la base de datos y datos iniciales
